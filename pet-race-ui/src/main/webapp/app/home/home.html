The Great Pet Monster Race

We are greeks! Well maybe we are not greeks, but we do make awesome Kubernetes demos.

This application is hosted on Kubernetes, with a backend running in Cassandra. GPMR was built to demostrate PetSets and other Kubernetes features, while running Cassandra at scale. This project is 100% open source.

Let the races begin.
